MarkDown语法中代码怎么换行？

问题：

MarkDown语法中明明有代码标签但是为什么代码的展示这么弱？连换行都不行？

回复：

不是markdown的语法弱没有准备代码标签，也不是前台展示代码渲染框架没有为代码显示做准备，是我们自己不了解……
其实百度的时候会发现，几乎没有答案……（除了我在贴吧里面问了这么个问题以外^_^）
这种现象只能反应两种情况

大家都知道根本不需要问
是一个很前卫的话题，几乎没有人了解
很显然，对于MarkDown语法中代码怎么换行的问题，应该更靠近前者。
发现代码不会换行的同学不出意料的话， 应该都是这么写的

在最一开始的三个点与上面的内容价格空格就行了
